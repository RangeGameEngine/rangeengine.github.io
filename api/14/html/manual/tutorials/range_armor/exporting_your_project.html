Exporting your Project

This tutorial will show you how to export properly a project using Range Armor, Be sure to follow all the steps to prevent fails and errors.

Setup The Scene

Setup the project to the right scene and camera, the standalone will start from the scene and camera that your project is.

Saving the file as Range Protected File (RASEC)

RASEC is the new file security system to Range Armor (Range Engine 1.4+)

After the scene is setup, you need the save your file as a .rasec

Tip

To organize better your project make a new folder and save the .rasec

Setup Range Armor

Now you need to open Range Armor, you can find it in File or Info tab

Warning

In Range Engine 1.2 or all Linux releases you need to open Range Armor in Range Engine folder RangeEngine/YourSystem/1.2/rangearmor/

Now you need to create a New Project

Tip

Create the project inside the folder you made to the .rasec file

Warning

Don’t try to create the project inside Program Files folder, otherwise it will fail

Importing Project, Dll’s and Player

Range Armor require some files inside the new project folder you create, as your project File, some engine dll files and folders, and the blenderplayer

When you create a Range Armor project, a folder is created with 4 folders in

The folders is:

  • Data (Your Project)

  • Engine (Engine files)

  • Icons (Custom Icons)

  • Launcher (Range Armor Files)

Data Folder

First of all, move your project .rasec to the Data folder and delete the Example Game.range

Engine Folder

Now you need to move some files from the Range Engine release version to the Engine folder

In this example the project will be exported to Windows x64, so you need to copy some Range Engine Windows x64 files to the Engine/Windows64 folder

Warning

If you want to export to Windows/Linux x86/x64 you got to have this Range Engine releases downloaded

Icons Folder

In this folder you can set custom icons to your project executable file, you just got to replace the stardard icons in the folder for your custom icons with the same name as the default icons

Setup The Project

Now go in Project panel in Range Armor, and do some settings

You need to change:

  • The project name

  • Version

  • .rasec file path (data folder)

Building the Data

Now you must to go in Tasks tab and compile the data

You must compile the data in order:

  • Build Data

  • Set Icons

Exporting

Finally you just have to choose the platform you want to export your project

Tip

If you want export to more platforms just choose All Platforms

When the export gets finished you can find you project in the Release folder

Conclusion

Now your project is properly exported, you already can publish your game in online platforms without problems with security and licensing!
