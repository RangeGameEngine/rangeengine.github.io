Changelog

RanGE 1.0

Principal Changes


  • Area Light

    • New Lamp Illumination Type.



  • Responsive Profiler

    • Best Engine Depuration.



  • Layer Weight

    • Now You Can Control The Layer Weight Of Each Layer, Allowing For More Complex Animations.



  • Character Physics Update

    • Character Physics Now Has Smooth Movement And Jump Direction.



  • Text Resolution

    • Change The In-Game Text Resolution (High Resolution Can Slow Performance)



  • Physics Culling Velocity

    • Objects With Physics Culling Only Stop Physics When They Are Motionless.



  • Object Physics Friction

    • New Function On Object Physics.



  • worldScalingObject Now Influence To obstacleSimulation

    • Now Obstacle Simulation Size Influences With Object Scaling, Great For Objects Instantiated With Different Sizes.



  • Reordering Components

    • You Can Arrange The Order Of Components In The Object.



  • Components Header

    • You Can Organize Your Components With Components Header.



  • New Mouse Events (Logic Bricks)

    • Mouse Side Buttons Are Now Available On Logic Bricks.



  • Repeat For In Delay Sensor

    • Number Of Times The Sensor Will Repeat Before Stopping.



    API, Bug Fixes And Much More!
