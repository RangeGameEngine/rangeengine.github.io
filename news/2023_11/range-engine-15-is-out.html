Range Engine 1.5 Is Out!

26 November 2023

We’re moving forward with some big changes! Range Engine 1.5 is finally among us and we can't believe you're going to let it pass you by. Work everyday with passion and determination is what makes Range Engine grow so fast and bring so many great features, so let us share with you what we've been working these last months.

Some new features:

  • New Sky Shader

    The Sky Shader got a big update, now there is sun in the sky and its strong light on the horizon, it also has Nadir and Ground Color.

  • Foliage Shader

    Perfect for leaves, grass or any object, Foliage Shader uses the GPU to simulate wind. It is extremely optimized and easy to configure.

  • Realtime Viewport Nodes

    Viewport now supports real-time viewing of nodes. It helps to improve development time as you can see the changes without needing to start the game.

  • Sound Updates

    Added full support for speaker object with new options to control the sounds. The audio library has also received a major update.

  • Support for ORM textures

    ORM textures consume less memory and are faster compared to traditional approaches as it has 3 textures combined in only one image.

  • Custom Mouse Cursor

    You can easily custom you in-game mouse cursor, with no workarounds needed.

  • Import Range

    import bge has been deprecated, it helps to change the engine identity and in some cases the scripts licensing.

See all the changes with gifs and examples in the Release Notes
