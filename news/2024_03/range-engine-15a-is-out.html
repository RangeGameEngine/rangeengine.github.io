Range Engine 1.5a Is Out

24 March 2024

Range Engine 1.5 was a sucessful release in it's early access, this release brought to the engine new great features and great stability, because of the on going development of the next release we decided to make more improvements and bug fixes to Range Engine 1.5 and finally bring this release for everyone!


Range Engine 1.5a is available today on Patreon and will have it's public release 04/15/2024.

As this is a ‘minor version’, it primarily focuses on enhancing stability, resolving bugs and crashes, and implementing minor changes to the interface and engine operation.


Some of these changes include:

  • New Shadow Filters
  • Joystick Input Events
  • Sky Shader Improvements / Sky Coordinates
  • Interface Icons Colors
  • Improved FXAA Filter

See all the changes with gifs and examples in the Release Notes


We also have good news about the development of the next Range Engine release which began it's development on December 2, 2023 and is now in the final phase of the OpenGL update, in addition to beginning the initial phase of the Renderer development. There's a lot to show about it so we decided to release an article dedicated only to this version soon, so stay tunned to the next steps of Range Engine!
