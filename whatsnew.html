What's New?

Take a look at some of the new Range features

Realtime Debug Mode

A fully reworked Debug Mode that allows
you to debug the profiling and manipulate objects in real time

PBR

(Physically Based Rendering)

Graphics rendering technique based on the physical principles of light and reflection, The goal is to produce realistic images that are consistent with the laws of physics, including the behavior of light on different surfaces and lighting conditions.

Range Armor Security

With Range Armor you can export your project with 3 steps security, You can also sell and distribute your project without licensing problems!

New Interface

One of the most important things to an engine is the interface, it needs to be comfortable and easy to use, The interface is constantly being re-designed to make the game development easy and efficient.

Python 3.10.9

Python is one of the most popular languages, which is very intuitive and easy to learn, Python 3.10.9 is already available in Range, and Python 3.11 soon!

Realtime Viewport

To make the game development more dynamic the engine need to be fast, we're working to make the viewport graphically more similar to the game mode porting new filters and making the nodes setup work in realtime.

Animation Events

Animation Events allow you to call functions in specific moments of an animation, Allowing you a much more simpler and organized coding development.

GameEngine DeltaTime

To avoid inconsistent actions with FPS variations, now Range Engine has the deltaTime function making the actions independent from framerate.

What's Next?

Some features we're planning and working on

OpenGL Update

A graphical API is a essential part of an engine, that's why we decided to start working on a update of the OpenGL, to make the engine more stable, optimized, and with new graphical features.

GPU Skinning

GPU skinning is a 3D animation technique that process the deformations of a model's bones on the GPU instead of the CPU. This can increase frame rate and improve animation performance in up to 90%

Cython

Cython is an optimising static compiler for Python and the extended Cython. It makes writing C extensions for Python as easy as Python itself.

Bricky Nodes

Bricky Nodes brings the actual nodes workspace with new and updated logic bricks, enabling you to make easy, complex and optimized logics.
